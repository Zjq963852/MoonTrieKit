# MoonTrieKit Roadmap

## 0.1.x 基础稳定期

- 完成词典构建、精确查询、前缀匹配、自动补全和 JSON 导出。
- 完成带权词条、排序补全、最长前缀匹配和 QueryTrace 查询解释。
- 保持 CLI 演示、README、CHANGELOG 和协作模板完整。
- 用确定性测试覆盖公开 API 的典型路径和边界路径。

## 0.2.x Trie 存储增强

- 将内部存储替换为节点表，提升大词典前缀查询性能。
- 增加删除词条、节点计数和压缩路径统计。
- 增加大小写归一化和 ASCII / Unicode 策略文档。

## 0.3.x 自动补全增强

- 增加最近使用排序和命中高亮。
- 增加 top-k 查询和分页补全。
- 增加命令面板、编辑器提示和配置键查询示例。

## 长期方向

- 保持核心库零平台依赖。
- 增加 benchmark，比较数组后端和节点后端性能。
- 沉淀为 MoonBit 文本检索和开发工具生态的基础组件。
